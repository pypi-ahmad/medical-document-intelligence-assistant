# Beginner Guide

## Goal
Understand medical documents with AI while keeping control local and safe.

## Workflow
1. Login or bootstrap admin account.
2. Upload documents from Upload Center.
3. Click process (OCR + extraction + indexing).
4. Ask questions in AI Chat.
5. Use Search for evidence snippets.
6. Open Timeline/Medication/Lab pages.
7. Generate doctor-visit report.
8. Review memory and clear when needed.

## How to Read AI Output
- "Extracted information" = grounded in your uploaded docs.
- "Educational background" = general non-diagnostic context.
- Citations show where evidence came from.

## Safety Reminder
For any medical decision, discuss with qualified healthcare professionals.
