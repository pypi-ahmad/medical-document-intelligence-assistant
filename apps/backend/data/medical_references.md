# Local Educational Medical Reference Notes

Hemoglobin, white blood cell count, platelet count, creatinine, sodium, potassium, ALT, and AST are laboratory markers commonly interpreted using lab-specific reference ranges. A value outside a printed range can still need context such as age, hydration status, medications, and known conditions.

Medication names may appear with dosage forms and frequencies such as daily, BID, TID, or q8h. Document extraction can identify mentions and apparent changes, but medication decisions must be validated by licensed clinicians.

Discharge summaries often include hospital course, diagnoses listed by treating teams, follow-up appointments, and warning signs. Educational summaries should preserve wording from the source document and avoid introducing diagnostic conclusions.

Radiology report impressions may contain differential language. Explanations in plain language should clarify terminology but must not be treated as diagnostic advice.
